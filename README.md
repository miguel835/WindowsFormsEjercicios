# Guías y ejercicios en Windows Forms (WF)

En este repositorio se encuentran guías de como hacer determinada funcionalidad en Windows Forms y 
ejercicios propuestos con sus respectivas soluciones. 

### ¿Cómo clono este repositorio?
```bash
git clone https://github.com/miguel835/EjerciciosPOO.git
```

## Tabla de contenido por tema

### 1. Controles y eventos de Windows Forms

### [Reloj digital](https://github.com/miguel835/WindowsFormsEjercicios/tree/master/RelojDigital)
Utilizando el control Timer de Windows Forms diseñar un formulario en el cual permita mostrar la hora 
actual y que con ayuda del Timer se vaya actualizando 

### [Abridor de apps](https://github.com/miguel835/WindowsFormsEjercicios/tree/master/Abridor%20de%20apps)
Guia de como usar el control Process de WF para que se abran determinada aplicación
del usuario que tenga instalado en su sistema operativo Windows

### [Variables](https://github.com/miguel835/WindowsFormsEjercicios/tree/master/Variables)
Con ayuda de la programación estructurada y la programación orientada a eventos crear una interfaz en 
la cual se pueda mostrar un MessageBox con un mensaje dado por el usuario, adicionalmente a esto 
agrege varios botones y dos textbox que permitan hacer la suma tipo normal, suma long/int 64, 
suma float, suma double, suma decimal. 

Por último un botón cuya funcionalidad va a ser devolver un booleano si un determinado boton esta habilidado
o no. 

### [Añadir elementos a un ComboBox](https://github.com/miguel835/WindowsFormsEjercicios/tree/master/A%C3%B1adirElementosAUnComboBox)
Guia de como añadir elementos a un comboBox 

### [Mostrar mensajes varias veces](https://github.com/miguel835/WindowsFormsEjercicios/tree/master/MostrarMensajesVariasVeces)
Guia de como mostrar un messageBox varias veces 

### [Numeros meses](https://github.com/miguel835/WindowsFormsEjercicios/tree/master/NumerosMeses)
Crear un formulario en la cual a traves del ingreso de un determinado número por un  textbox este pueda
mostrar el nombre del mes, y viceversa si el usuario ingresa un determinado mes en otro textbox que 
este le devuelva el número del mes

### [Operadores principales](https://github.com/miguel835/WindowsFormsEjercicios/tree/master/OperadoresPrincipales)
En un formulario agregar los controles necesarios para que se permita el ingreso de dos números en dos textbox
diferentes y que a través de botones el usuario pueda seleccionar la operación básica que desee realizar y 
el resultado lo muestre en un textbox.

Se pide que además se agrege otros dos textbox que permita el ingreso de dos números y adicionar 
dos botones cuya funcionalidad va a ser verificar si el primer número es mayor, y verificar si el primer
número es igual que el segundo número, esto respectivamente. 

Utilizar el paradigma de programación orientada a eventos

### [Calculadora](https://github.com/miguel835/WindowsFormsEjercicios/tree/master/Calculadora)
Con Windows Forms cree una calculadora que permita realizar operaciones básicas, puede guiarse en el 
diseño tomando en cuenta la calculadora nativa de Windows 

### [Dialog_pro](https://github.com/miguel835/WindowsFormsEjercicios/tree/master/Dialog_pro)
Guía de como usar los cuadros de dialogos de Windows Forms con son: colorDialog, folderBrowserDialog,
fontDialog, openFileDialog, saveFileDialog

OJO: Aunque no se enseña como guardar en el saveFileDialog

### [ComboBox avanzado](https://github.com/miguel835/WindowsFormsEjercicios/tree/master/ComboBox%20avanzado)
Guia de como usar el comboBox para agregar elementos, buscar elementos, borrar todo el comboBox, 
como se obtiene el indice de un elemento seleccionado por el usuario en el comboBox, como contar
todos los elementos de un comboBox

### [Edad del usuario basado en fecha nacimiento](https://github.com/miguel835/WindowsFormsEjercicios/tree/master/EdadUserBasadoEnFechaNacimiento)
En WF hacer la lógica de un programa que permita calcular la edad del usuario basado en la fecha de nacimiento, 
es decir se debe diseñar la UI que permita que el usuario ingrese su año de nacimiento 

### [Eventos](https://github.com/miguel835/WindowsFormsEjercicios/tree/master/Eventos)
Guia de como usar los eventos en Windows Forms y los eventos que encontraras aquí son los más importantes
y utilizados.

TIP: Cuando lees el nombre de su evento literalmente te dice cuál es su función

### [MDI proyecto](https://github.com/miguel835/WindowsFormsEjercicios/tree/master/MDI_proyecto)
Guía de como usar los formulario MDI (Multiple Document Interface), viendo el formulario padre y 
sus formulario hijos.

### 2. POO en Windows Forms

### [Información Computador](https://github.com/miguel835/WindowsFormsEjercicios/tree/master/InformacionComputador)
Crear un programa que permita almacenar la información de un computador. 
El entorno gráfico deberá tener opciones para registrar las partes principales del computador: 
monitor, memoria RAM, Disco duro y procesador. 
Cada parte debe tener sus propiedades específicas. 
El usuario debe poder ingresar o editar las propiedades de cada parte y del computador.
Crear clases usando composición que gestionen estas propiedades.
El entorno gráfico debe ser amigable con el usuario.
Usar controles contenedores para diferenciar entre una parte u otra del computador

### [Codigo de colores de las resistencias](https://github.com/miguel835/WindowsFormsEjercicios/tree/master/CodigoDeColoresDeLasResistencias)
Con Windows Forms realizar un programa que permita calcular el código de colores de las resistencias, ten
en cuenta que el usuario debe de elegir los valores para su calculo y el cálculo debe de ser inmediatamente
cuando selecciona un valor.
Utilizar POO y se permite crear una sola clase Resistor, no más.
Utilizar la librería Drawing para graficar la resistencia

### 3. Libreria IO

### [Archivos de textos IO](https://github.com/miguel835/WindowsFormsEjercicios/tree/master/ArchivosDeTextosIO)
Guía de como usar un archivo de texto con ayuda de System.IO.File, es decir, como leer(cargar los datos) 
en un archivo de texto y como guardar en una dirección del sistema

### [Archivos IO](https://github.com/miguel835/WindowsFormsEjercicios/tree/master/ArchivosIO)
Guia de como usar el System.IO.File para copiar un archivo de un directorio a otro, como verificar la 
existencia de un archivo, como borrar un archivo, como mover el archivo de un directorio a otro y finalmente
como cortar el archivo

### [Directorios IO](https://github.com/miguel835/WindowsFormsEjercicios/tree/master/DirectoriosIO)
Guia de como usar el System.IO.Directory para crear una carpeta, borrar una carpeta, mover una carpeta y
verificar la existencia de una carpeta

### [Guía telefónica](https://github.com/miguel835/WindowsFormsEjercicios/tree/master/Gu%C3%ADaTelef%C3%B3nica)
En este ejercicio diseñe un formulario tipo MDI en el cual tenga un formulario padre y cuatros formularios hijos 
en el formulario padre agrege el control toolStrip en la cual tenga en cada botón tenga un estilo Imagen y Texto
en el primer botón será para llamar a un formulario hijo para el registro de datos, el segundo botón será
para llamar a un formulario hijo para realizar la búsqueda de datos por ID, el tercer botón deberá hacer 
una llamada al sistema Windows y llamar a la calculadora nativa del sistema, en el cuarto botón será para 
llamar a un formulario hijo que muestre información sobre el programa (Nombre, versión, año de creación, etc)
y finalmente se pide un último botón que será para cerrar el programa con this.Close().

Se pide además que configure el formulario para que el usuario pueda realizar la llamada de un formulario hijo
con ayuda del evento KeyDown, ejm: F2 muestra el formulario hijo de registro de datos, F3 Buscar, ESC Salir, etc.

Se solicita que se utilice la librería IO para guardar los datos con el nombre del ID + el dato a guardar 
en diferentes archivos de textos, es decir un archivo en el que se guarde el nombre, otro archivo para el apellido, etc.
Programe para que pueda realizar la buscar en el sistema del archivo con ayuda del ID escito por el usuario en un
textbox, además también valide que primero los archivos existan en el sistema para evitar que el programa se 
caiga. 

Finalmente modifique la información de ensamblado del programa para que se puedan visualizar en 
las propiedades el .exe información como nombre, versión del programa, empresa, etc. 

### 4. Base de dato PostgreSQL

### [Biblioteca PostgreSQL](https://github.com/miguel835/WindowsFormsEjercicios/tree/master/Biblioteca-PostgreSQL)
Crear un programa que permita acceder a una tabla de una base de datos, para ingresar nuevos registros y para 
consultar los datos guardados.
Para realizar este proceso deberá crear una base de datos que almacene información sobre datos de libros de una
biblioteca, tales como nombre del libro, autor, editorial y año de publicación.
Además, se deberá crear el entorno grafico que permita la interacción de los usuarios con la información.
Dentro del código necesario para el acceso a la base de datos se solicita crear una clase que permita 
almacenar nueva información y para consultar la registrada en la base de datos. Esta clase deberá tener
los métodos que permitan almacenar nueva información y consultar información almacenada
